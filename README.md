# GenAI Google Ads Performance Analyzer

## 📌 Overview
This project analyzes Google Ads campaign data and uses Generative AI to provide insights and recommendations.

## 🚀 Features
- Reads campaign data from CSV
- Calculates key metrics (CTR, CPC, Conversion Rate)
- Identifies top and worst performing campaigns
- Uses AI to generate insights and recommendations
- Interactive Q&A with your ad data

## 🛠️ Tech Stack
- Python
- Pandas
- Google Gemini API

## ▶️ How to Run

1. Clone repo
2. Create virtual environment
3. Install dependencies:
   pip install pandas google-generativeai python-dotenv
4. Add your API key in `.env`
5. Run:
   python main.py

- Which campaign is performing best?
- Why are conversions low?
- How can I improve ROI?
